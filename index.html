About Reminders

How it works

Enter your name once when adding a reminder. The app remembers you and filters to your reminders by default every time you open it. Your partner will see their own reminders by default — but either of you can switch to see everyone's.

Shared GitHub file

Set up GitHub sync in the add modal. On startup the app pulls the shared reminders.json and merges it with local data. Every new reminder you add is pushed back automatically. Both of you see all reminders; the author name shows who added what.

Filtering

  • Quick chips under the header — tap to switch.
  • Filter button (☰) — browse by person or group.
  • Your last filter is remembered between sessions.

Checking off & deleting

  • Tap the circle to mark done.
  • Tap ✕ on a pill to delete it.

Export / Import

Export downloads a .json backup. Import merges by ID — no duplicates created.
