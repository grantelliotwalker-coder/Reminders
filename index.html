About Reminders

What is this?

Reminders is a lightweight, offline-first reminder app that lives in a single HTML file. No account needed — everything is saved in your browser.

Colour · Priority guide

Groups

Assign any reminder to a group (Work, Personal, etc.). Use the filter button ☰ in the header or the quick chips below the header to filter your view.

Checking off & deleting

  • Tap the circle on a pill to mark it done.
  • Tap the ✕ on a pill to delete it.

Export & Import

Use ⬇ Export in the footer to save your reminders as a JSON file. Use ⬆ Import to load them back on any device.

GitHub Sync

Enter your GitHub token + repo details in the settings below the add form to read/write reminders directly to a JSON file in a repo. Reminders are merged by ID so no duplicates.

Your name

Enter your name once in the add modal — it's saved locally and stamped on every reminder you create, making shared repos easy to track.
